Pequeno script feito em python para a remocao de caracteres especiais do HTML
Exemplo:
    '&ccedil;' -> 'ç',
    '&aacute;' -> 'é',,,
-----------------------------------------------------------------------------
Dependecias:
    Python 2.7.x,
    htmlentity2ascii
